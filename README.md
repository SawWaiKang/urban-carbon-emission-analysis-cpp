# Urban Carbon Emission Analysis (C++)

## Overview

This project is a Data Structures and Algorithms assignment developed in C++.

The system compares the performance of Arrays and Singly Linked Lists by processing urban transportation datasets from three different cities.

## Features

- Array implementation
- Singly Linked List implementation
- Carbon emission analysis
- Age group analysis
- Sorting algorithms
  - Bubble Sort
  - Quick Sort
  - Merge Sort
- Searching algorithms
  - Linear Search
  - Binary Search
- Execution time comparison
- Memory usage comparison
- Console-based interface

## Language

- C++
- Standard Template Library (STL)
- Object-Oriented Programming (OOP)

## Datasets

- City A – Metropolitan City
- City B – University Town
- City C – Suburban/Rural Area
